## Contributing

Articles must be proposed via pull request and adhere to the desiderata below.

### Desiderata

- Articles should address broad domains — technology, governance, capital markets, social systems — rather than promoting specific products, services or agendas
- Commentary should be grounded in evidence, research or rigorous reasoning
- Interdisciplinary perspectives are encouraged — the best work draws from multiple fields
- Articles should advance a clear argument or insight, not merely summarise existing work
- References should be included where claims are made
- No promotional content, sponsored posts or commercial solicitation
